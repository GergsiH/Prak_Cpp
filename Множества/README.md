# Проект: Реализация класса множеств на C++

В этом проекте я разработала класс для работы с математическими множествами целых чисел, реализующий основные теоретико-множественные операции с эффективными алгоритмами работы.

## Что я реализовала

### 1. Класс Set с динамическим управлением памятью
Создала класс `Set`, хранящий массив целых чисел с динамическим выделением памяти. Реализовала конструкторы по умолчанию, копирования и деструктор с корректным освобождением ресурсов. Добавила сортировку элементов внутри множества через статический метод `sort` для обеспечения уникальности операций.

### 2. Основные теоретико-множественные операции
Реализовала три фундаментальные операции: объединение (`OR`), пересечение (`AND`) и разность (`DIFF`) множеств. Для пересечения использовала двойной цикл с предварительной сортировкой, для разности - алгоритм с флагом проверки наличия элемента во втором множестве.

### 3. Операции добавления и удаления элементов
Добавила методы `ADD` и `KICK` для модификации множества с проверкой уникальности элементов через бинарный поиск. Реализовала функцию `FoundElem` для быстрого поиска элемента в отсортированном массиве методом деления пополам.

### 4. Перегрузка операторов ввода/вывода
Переопределила операторы `<<` и `>>` для удобной работы с множествами через стандартные потоки. Формат ввода поддерживает последовательность чисел с терминатором `-1`, что позволяет гибко задавать множества произвольного размера.

## Особенности реализации и технологии
Проект написан на C++ с акцентом на ручное управление памятью через `new`/`delete`. Класс использует фиксированный размер массива `M = 50`, но включает базовую логику для потенциального расширения. Все операции поддерживают сортировку элементов для обеспечения консистентности данных. Интерфейс включает интерактивное меню с шестью операциями и обработкой ошибочных команд. Алгоритмы оптимизированы для работы с отсортированными данными, что улучшает производительность операций поиска и пересечения.
