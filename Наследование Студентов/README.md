# Проект: Система управления студентами и аспирантами на C++

В этом проекте я разработала систему для управления учебными группами, реализующую иерархию классов студентов и аспирантов с полным набором операций для работы с группами.

## Что я реализовала

### 1. Иерархия классов студентов
Создала базовый класс `Student` с данными о студенте (имя, ID, средний балл) и производный класс `Graduates` для аспирантов с дополнительной информацией о дипломной работе и научном руководителе. Реализовала конструкторы, деструкторы и операторы копирования/перемещения с корректным управлением памятью.

### 2. Абстрактный интерфейс для групп
Разработала абстрактный класс `Group` с виртуальными методами для добавления/удаления студентов, вывода информации и доступа к данным по индексу. Создала две конкретные реализации: `StudGroup` для обычных студентов и `GradGroup` для аспирантов.

### 3. Управление группами студентов
Реализовала полный набор операций для работы с группами: добавление студентов через консольный ввод, удаление по ID, вывод всей информации о группе. Добавила проверку на максимальное количество студентов в группе и обработку ошибок при поиске несуществующих ID.

### 4. Операции копирования и перемещения
Написала конструкторы копирования и перемещения, а также соответствующие операторы присваивания для безопасной работы с динамической памятью. Реализовала правильное освобождение ресурсов в деструкторах всех классов.

## Особенности реализации и технологии
Проект написан на C++ с использованием принципов объектно-ориентированного программирования (наследование, полиморфизм, инкапсуляция). Для работы с динамической памятью применяется ручное управление через `new`/`delete` с обработкой всех случаев утечек. Система использует массив указателей фиксированного размера для хранения студентов, реализует перегрузку операторов ввода/вывода и индексации. Код включает обработку исключений при доступе по несуществующему ID и обеспечивает безопасную работу с C-строками через динамическое выделение памяти с ограничением длины.
